  2. Question: Define version control and give one example.

    Answer: Version control is a system of controling and keeping record of changes made to documents, websites, software etc. One Example of a version control system is within Github, which tracks the changes made to user files in their database.

  3. Question: What tag is used to create the title of a web page? Give an example.

    Answer: The <title> tag designates the title of a webpage in the metadata and displays this title in the browser's navgation tab.

  4. Question: What tag is used to create a paragraph in html? Give an example.

    Answer: The <p> tag creates a paragraph for text. Example: <p>This is a paragraph!<p>

  5. Question: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Answer: The tags <span>, <p> and <i> are all used to display text and the user cannot interact with their content. The <a> tag however, creates a link the user can interact with and it can be displayed as a number of other visual elements, such as images or buttons.

  6. Question: What are the tags required to create a navigation list in HTML?

    Answer: To create a navigation list use a <nav> tag to create the container designating this section is for navigation elements, then within the <nav>, use a <ul> tag to create a list and <li> to create list items. The code should look like this when put together:
    <nav>
      <ul>
        <li></li>
        <li></li>
        <li></li>
      </ul>
    </nav>

  7. Question: What element is used to create images? Name its attributes.

    Answer: The <img> tag displays an image. The height and width attributes can be applied to the tag to determine the image's displayed size and the alt attribute can be applied to provide a text tied to the image.

  8. Question: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Answer: The tags <table>, <td> and <tr> are all used to create and customize tables. The <b> tag does not manipulate tables, it boldens text.

  9. Question: Give an example of color picker input.

    Answer: The <input type="color"> tag allows the user to select a custom colour.

  10. Question: What is the use of name and value attributes? Give two examples

    Answer: The name attribute assigns a name to a tag, which can them be used to call the tag in Javascript. Example: <button name="submit" type="submit">submit</button>
    The value attribute assigns a value to a tag. Example: <input type="text" id="first-name" name="first-name" value="Jean"> The input box will display and input Jean until the user inputs a different value.

  11. Question: Give an example of inline style.

    Answer: An inline style is a style attribute applied directly to an HTML element. Example: <p style="color:red;">Red text!</p> = Red text!

  12. Question: How would you link an external style sheet? Give example.

    Answer: To link CSS to an HTML document you have to add a <link> tag to the head section of the document.
